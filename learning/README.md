# learning/ — instructor-managed (don't edit)

This folder belongs to the **course instructor** (Claude Code). It is created/updated automatically as
you progress. You normally never need to touch anything in here.

- **`_progress.md`** — your saved session state (course mode, active plan, completed/current feature,
  exact next step, demo folders). Created on first run; updated after every feature and on pause.
- **`_notes.md`** — the instructor's running per-feature teaching notes.

These `_`-prefixed state files are **git-ignored** so your personal progress stays local and never gets
committed. This folder stays in the repo via this README.

➡️ Start or continue from the repo root with `/start-learning` or `/resume-learning`.
